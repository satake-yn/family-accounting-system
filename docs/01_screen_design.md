## 2. 入力画面 (Input Screen)

### 2.1 画面レイアウト (iPhone)

```
┌──────────────────────┐
│ 家計簿           ▼   │  ← 月選択ドロップダウン
├──────────────────────┤
│                      │
│ 収入 / [支出] ◀▶    │  ← セグメント切り替え
│                      │
├──────────────────────┤
│ 日付                 │
│ [2026-07-23] ▼      │  ← 日付ピッカー
├──────────────────────┤
│ 金額                 │
│ [____________]       │  ← フォーカス時キーボード
│ ¥ 0                  │  ← リアルタイム表示
├──────────────────────┤
│ カテゴリ             │
│ [選択してください ▼]  │  ← 必須
├──────────────────────┤
│ 店舗                 │
│ [入力/候補から選択]   │  ← 候補表示
├──────────────────────┤
│ タグ                 │
│ [+タグを追加]        │  ← 複数追加可能
│ ✕ タグ1 ✕ タグ2    │
├──────────────────────┤
│ メモ                 │
│ [メモを入力...] ▼   │  ← 折りたたみ
├──────────────────────┤
│ [   保存   ] [破棄]  │
└──────────────────────┘
```

### 2.2 入力項目仕様

| 項目 | 必須 | 型 | 初期値 | 入力例 | 備考 |
|------|------|-----|--------|--------|------|
| 収支 | ✓ | 列挙 | 支出 | 収入/支出 | セグメント |
| 日付 | ✓ | 日付 | 当日 | 2026-07-23 | 変更後保持 |
| 金額 | ✓ | 正整数 | 空 | 12500 | 3桁カンマ |
| カテゴリ | ✓ | 階層選択 | 空 | 食費 > スーパー | ドロップダウン |
| 店舗 | ☐ | テキスト | 空 | オーケー | オートコンプリート |
| タグ | ☐ | 複数選択 | 空 | 旅行, 帰省 | チップ形式 |
| メモ | ☐ | テキスト | 空 | 日用品含む | 折りたたみ |

### 2.3 ユーザーフロー

**シナリオ**: 支出を30秒で入力する

```
1. アプリ起動
   ↓
2. 日付が前回設定日付で保持されている
   ├─ OK: 次へ
   └─ NG: 日付ピッカーで変更 (→ 保持)
   ↓
3. 「支出」を選択 (デフォルト)
   ↓
4. 金額を入力 [1250]
   ↓
5. カテゴリを選択 [食費 > スーパー]
   ↓
6. 店舗候補から選択またはスキップ
   ↓
7. 「保存」クリック
   ↓
8. 入力画面リセット (金額・カテゴリクリア、日付は保持)
   ↓
9. 次の取引入力へ (→ 4. へ)
```

**目標**: 30秒以内に完了

### 2.4 キーボード最適化 (iPhone)

```
金額入力時:
  ┌─────────────────────┐
  │ [1250]              │
  ├─────────────────────┤
  │  1 2 3 4 5 6 7 8 9 │
  │  0 . ← ✓            │
  └─────────────────────┘

表示:
  ┌─────────────────────┐
  │ ¥12,500            │
  │ 入力中...           │
  └─────────────────────┘
```

### 2.5 バリデーション

```
金額:
  ✓ 正整数のみ
  ✓ 0 より大きい
  ✗ 負数は不可
  ✗ 小数点は不可

カテゴリ:
  ✓ 必須選択
  ✗ 空は保存不可

日付:
  ✓ 当日〜過去1年以内
  ✗ 未来日付は不可
```

## 3. 履歴画面 (History Screen)

### 3.1 画面レイアウト

```
┌──────────────────────┐
│ 履歴          🔍 ⚙️  │
├──────────────────────┤
│ [全期間▼] [すべて▼]  │  ← フィルタ
├──────────────────────┤
│ 2026-07-23           │
│ ├─ 食費/スーパー     │
│ │  ¥12,500          │
│ │  オーケー         │
│ │  [編集] [複製] [削除]
│ │                    │
│ ├─ 飲食/外食        │
│ │  ¥8,900           │
│ │  渋谷の店         │
│ │  [編集] [複製] [削除]
│ │                    │
│ 2026-07-22           │
│ ├─ 給料/給与        │
│ │  ¥ 300,000        │
│ │  [編集] [複製] [削除]
│                      │
└──────────────────────┘
```

### 3.2 フィルタ機能

| フィルタ | 種類 | 例 |
|----------|------|-----|
| 期間 | 日付範囲 | 全期間, 今月, 先月, 今年 |
| カテゴリ | 複数選択 | すべて, 食費, 衣服 |
| 店舗 | テキスト検索 | オーケー |
| タグ | 複数選択 | 旅行, 帰省 |
| 金額範囲 | スライダー | ¥0 ～ ¥50,000 |

### 3.3 操作

- **編集**: トランザクション編集画面へ遷移
- **複製**: 同一内容で新規入力画面を起動
- **削除**: 論理削除確認ダイアログ表示

## 9. アクセシビリティ

- コントラスト比: 4.5:1 以上
- フォントサイズ: 最小 16px (本文)
- タッチターゲット: 最小 44x44px
- キーボード操作対応
- スクリーンリーダー対応

## 10. パフォーマンス指標

| 指標 | 目標 |
|------|------|
| LCP (Largest Contentful Paint) | < 2.5s |
| FID (First Input Delay) | < 100ms |
| CLS (Cumulative Layout Shift) | < 0.1 |
| 入力から保存完了まで | < 2s |
