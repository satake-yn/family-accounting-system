# Family Accounting System

> **30秒で入力し、5分で振り返る。10年以上使える家計簿。**

## 📋 プロジェクト概要

Google フォーム + Google スプレッドシートによる家計簿を全面刷新し、入力性・保守性・拡張性を向上させた Web アプリケーション。

### システム構成

```
iPhone/PC/iPad
      ↓
PWA (HTML/CSS/JavaScript)
      ↓
Google Apps Script
      ↓
Google Spreadsheet (Database)
```

- **完全無料**で運用
- **iPhone最優先**設計
- **夫婦共有**対応
- **10年以上運用**を想定

## 📋 必須画面

- **入力** - 30秒で家計簿入力
- **履歴** - トランザクション管理
- **集計** - 月別/年別/カテゴリ別分析
- **資産** - 銀行・証券等の資産推移
- **設定** - マスタ管理・データ移行

---

## 🏗️ 設計原則

1. **入力を最優先する**
2. **必須項目を増やさない**
3. **タグで表現できるものはタグで管理する**
4. **スプレッドシートはDBであり、UIはアプリに集約する**
5. **集計データは保存せず、その都度計算する**
6. **長期間運用しても保守しやすい構造を維持する**

---

## 📝 コンテキスト

このプロジェクトは「趣味のコード」ではなく、**小規模業務システム**として扱われます。

設計品質を最優先とし、実装前に画面設計・DB設計・API設計を完成させます。

これにより、将来どのAIや開発者が引き継いでも、品質の高い成果物を作成できる状態を目指します。
